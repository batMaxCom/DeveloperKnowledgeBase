# Cron
Синтаксис:

Поле			Допустимые значения
мин (минуты)		0–59
час (часы)		0–23
дм (день месяца)	1–31
мес (месяц)		1–12
дн (день недели)	0–7
Звездочка (*) означает «все» или «любой».

мин час дм месяц день пользователь команда
00  23  *  *     1-5  root         /sbin/shutdown -h now

Для интерактивного режима:
sudo crontab -e
