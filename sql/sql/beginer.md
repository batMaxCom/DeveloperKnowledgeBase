# Типы БД

## Реляционные (SQL)
`База данных (БД)` — это организованное хранилище данных, к которому можно быстро обращаться, изменять, структурировать и анализировать информацию.

Характеристики:

- таблицы, колонки, связи
- строгие правила целостности
- транзакции

мощный язык запросов (SQL)
## Нереляционные (NoSQL)

Документы, ключ-значение, графы, колонки.

Используются там, где нужна большая гибкость или огромный масштаб.

# СУБД
СУБД — система управления базами данных.

Она:
- хранит данные;
- обрабатывает запросы;
- обеспечивает целостность;
- контролирует транзакции;
- управляет доступом.

# Что такое SQL?

SQL — язык запросов к реляционной БД.

## Большие группы команд:

### 1. DQL — Data Query Language

Язык запросов данных.
Используется для получения информации из базы.

Главная команда:
- SELECT — выборка данных

### DML — Data Manipulation Language

Язык манипуляции данными.
Работает с содержимым таблиц.

Команды:

- INSERT — вставка записей
- UPDATE — изменение данных
- DELETE — удаление данных

### DDL — Data Definition Language

Язык определения данных.
Отвечает за структуру таблиц и схем.

Команды:
- CREATE — создание таблиц, схем, объектов
- ALTER — изменение структуры
- DROP — удаление объектов

### DCL — Data Control Language

Язык управления доступом.
Настройка прав пользователей.

Команды:
- GRANT — выдать права
- REVOKE — отозвать права
